Schritt-für-Schritt-Anleitung
← System ändern: –
1
Im WordPress-Backend anmelden
Rufen Sie ihrepraxis.de/wp-admin auf und melden Sie sich an.
2
Kontakt- oder Patientenseite öffnen
Links im Menü: Seiten → Klicken Sie auf die Seite, auf der der Hinweis erscheinen soll (z. B. „Kontakt").
3
HTML-Block einfügen
Gutenberg (neu): Klicken Sie auf + → suchen „HTML" → „Benutzerdef. HTML" wählen.
Classic Editor (alt): Oben auf den Reiter „Text" wechseln (nicht „Visuell").
4
Code einfügen & speichern
Wählen Sie unten einen Stil, kopieren Sie den Code, fügen Sie ihn ein. Dann auf „Aktualisieren" klicken.
1
TYPO3-Backend öffnen
Rufen Sie ihrepraxis.de/typo3 auf und melden Sie sich an.
2
Seite im Seitenbaum wählen
Links im Seitenbaum auf die Seite klicken, auf der der Link erscheinen soll.
3
Neues Inhaltselement: „Reines HTML"
Klicken Sie auf + Inhalt hinzufügen → Reiter „Spezielle Elemente" → „Reines HTML".
4
Code einfügen, speichern, Cache leeren
Code ins HTML-Feld einfügen → Speichern → oben rechts ⚡ „Alle Caches leeren".
1
Wix Dashboard öffnen
Auf wix.com anmelden → „Website bearbeiten" klicken.
2
HTML-Element einfügen
+ Hinzufügen → Einbetten → HTML-Code einbetten auswählen und auf der Seite platzieren.
3
Code eingeben
Doppelklick auf das Element → „Code eingeben" → Code einfügen → „Aktualisieren".
4
Veröffentlichen
Oben rechts auf „Veröffentlichen" klicken – fertig.
Hinweis: Im Wix Free-Tarif kann das HTML-Embed eingeschränkt sein. Der einfache Text-Link funktioniert immer.
1
Direkt auf Ihrer Seite anmelden
Rufen Sie Ihre Website auf, melden Sie sich an und klicken Sie auf „Bearbeiten".
2
HTML-Widget hinzufügen
Klicken Sie zwischen zwei Elementen auf + → „Weitere Inhalte" → „HTML/CSS/JavaScript".
3
Code einfügen & speichern
Code ins Feld einfügen → auf Speichern klicken.
Einfacher: Text markieren → Link-Symbol in der Toolbar → URL: https://patienten.dgpar.de
1
Joomla-Backend öffnen

2
Artikel öffnen

3
Quellcode-Ansicht öffnen

4
Code einfügen & speichern
Code einfügen → OK → „Speichern & Schließen".
1
HTML-Datei im Editor öffnen
Öffnen Sie die .html-Datei in einem Texteditor oder fragen Sie Ihren Entwickler.

Code an passender Stelle einfügen

Lang · FAQ / Patienteninformation
Warum erhalte ich eine Rechnung von dgpar? Unsere Praxis hat die privatärztliche Abrechnung an dgpar übertragen. dgpar ist ein zertifizierter Abrechnungsdienstleister und verwaltet Ihre Daten DSGVO-konform. Auf patienten.dgpar.de können Sie Ihre Rechnungen jederzeit einsehen, herunterladen und bei Fragen direkt Kontakt aufnehmen – kostenlos und sicher.
